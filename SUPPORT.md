# Support

Thanks for using **FSLED Academy**. This page explains where to get help and how to get the fastest, most useful response.

---

## Before You Ask

1. Read the [README](./README.md) — installation, tech stack, and getting-started steps.
2. Check the [ROADMAP](./ROADMAP.md) — the feature you want may already be planned.
3. Search [existing issues](https://github.com/khaaliswooden-max/fed_sled_academy/issues) and [discussions](https://github.com/khaaliswooden-max/fed_sled_academy/discussions) — someone may have already answered your question.

---

## Where to Go

| I want to… | Use this |
|------------|----------|
| Ask a question or share an idea | [GitHub Discussions](https://github.com/khaaliswooden-max/fed_sled_academy/discussions) |
| Report a bug in the application | [Bug Report issue](https://github.com/khaaliswooden-max/fed_sled_academy/issues/new?template=bug_report.md) |
| Report a factual error in course content | [Content Error issue](https://github.com/khaaliswooden-max/fed_sled_academy/issues/new?template=content_error.md) |
| Request a feature | [Feature Request issue](https://github.com/khaaliswooden-max/fed_sled_academy/issues/new?template=feature_request.md) |
| Report a security vulnerability | **Do not open an issue** — follow the [Security Policy](./SECURITY.md) |
| Contribute code or content | See [CONTRIBUTING](./CONTRIBUTING.md) |

---

## Response Expectations

FSLED Academy is maintained by [Visionblox LLC](https://visionblox.com) alongside client work. We aim to:

- Triage new issues within **5 business days**
- Respond to discussions as time allows
- Acknowledge security reports within **48 hours** (see [SECURITY.md](./SECURITY.md))

These are targets, not guarantees. Clear, reproducible reports get resolved fastest.
