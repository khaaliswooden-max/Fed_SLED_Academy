## [1.0.0] — 2026-02-24

### Added
- Initial release of FSLED Academy
- 8 complete courses (FSLED-090 through FSLED-699)
- XP-gated course progression system
- 8-tier rank system: Recruit → Capture Legend
- Mission briefing format with key intelligence panels
- Field Exercise quiz system with 5 questions per course
- Pass threshold of 80% required to advance
- Detailed explanations for all quiz answers
- Persistent XP and completion state via browser storage API
- Terminal/command center aesthetic with IBM Plex Mono and Orbitron typography
- Color-coded course tracks with individual visual identity
- Notification system for XP awards and rank progression
- Course completion bonus XP
- FSLED-699 Capstone: full CISA ZTA capture simulation (Operation IRONGATE)
