# FSLED Academy — Product Roadmap

This roadmap reflects the planned development trajectory. Items are subject to change based on community feedback and contributor availability.

---

## Vision

FSLED Academy becomes the standard onboarding and continuous learning system for federal and SLED capture professionals — used by BD teams, consultancies, and training programs to build durable acquisition knowledge through practice, not passive reading.

---

## Release Milestones

### v1.1 — Content Expansion
*Target: Q2 2026*

- [ ] Expand all courses to 3+ missions with full lesson content (401, 501, 601 currently have abbreviated content)
- [ ] Add 10+ questions per Field Exercise (up from 5) with randomized question pools
- [ ] Add FSLED-090 prerequisite glossary — standalone intel card vocabulary bank
- [ ] Add course-level review: summary of what was learned after each completion

### v1.2 — SLED Track
*Target: Q3 2026*

- [ ] FSLED-110: SLED Procurement Fundamentals (vs. federal: key differences)
- [ ] FSLED-120: Cooperative Contracts & Purchasing Vehicles (NASPO, OMNIA, Sourcewell, etc.)
- [ ] FSLED-130: State-Specific Procurement Deep Dives (CA, TX, WA, MD, VA, FL, MT)
- [ ] FSLED-140: Education Sector Procurement (K-12 E-Rate, higher ed procurement offices)

### v1.3 — Intel Card Mode
*Target: Q3 2026*

- [ ] Flashcard system for key terms and concepts
- [ ] Spaced repetition logic for review scheduling
- [ ] Vocabulary test mode (match term to definition)
- [ ] Card sets exportable as PDF study sheets

### v2.0 — Platform Features
*Target: Q4 2026*

- [ ] User account system with persistent server-side progress
- [ ] Team/cohort mode — share progress across a BD team
- [ ] Leaderboard — ranked by XP within a team or globally
- [ ] Instructor dashboard — assign courses, track completion, export reports
- [ ] Certificate generation on course or curriculum completion
- [ ] Mobile-responsive layout optimized for iOS and Android browsers

### v2.1 — Simulation Expansion
*Target: Q1 2027*

- [ ] FSLED-699 branching outcomes — different strategic choices in Operation IRONGATE lead to different scenario branches and outcomes
- [ ] New Capstone scenarios: DoD logistics IDIQ, HHS health IT GWAC, state transportation IT
- [ ] After-action review mode — replay decisions and see alternate outcomes
- [ ] Timed challenge mode — simulate real proposal deadline pressure

### v2.2 — AI Integration
*Target: Q2 2027*

- [ ] AI-generated scenario variations for the Capstone — new scenarios generated from real SAM.gov data patterns
- [ ] Proposal critique mode — submit a sample proposal section for structured feedback
- [ ] Personalized learning path — adaptive course sequencing based on assessment results
- [ ] Smart question difficulty scaling based on learner performance history
